Galvenā HTML CSS Javascript

Kas ir HTML?

HTML (Hypertext Markup Language) ir marķēšanas valoda, kas tiek izmantota tīmekļa lapu struktūras izveidei. Tā izmanto elementus un tagus, lai aprakstītu dažādas satura daļas, piemēram, virsrakstus, rindkopas, attēlus un saites.

Kas ir CSS?

CSS (Cascading Style Sheets) ir stilu lapa, kas tiek izmantota, lai noformētu HTML dokumentus. Tā nosaka izkārtojumu, krāsas, fontus un citus dizaina elementus, padarot tīmekļa lapas pievilcīgākas un pielāgotas lietotāja vajadzībām.

Kas ir JavaScript?

JavaScript ir programmēšanas valoda, kas tiek izmantota tīmekļa vietņu interaktivitātes pievienošanai. Tā ļauj izveidot dinamisku saturu, animācijas, formu validāciju un daudz ko citu, padarot lapas interaktīvas un lietotājam draudzīgas.
